SheetSnap: Table OCR

SheetSnap extracts table data from screenshots, scans, and photos on macOS. If you need help with importing files, model download behavior, export issues, or extraction quality, contact support below.

What to include in a support request

  • The macOS version you are using.
  • A short description of the problem.
  • The image or table type you were trying to extract.
  • A screenshot of the app state or error message, if available.

Common questions

Does the app work offline?
After the model asset has been downloaded on first launch, the app can continue working locally with previously available assets.

What files can I import?
Common image formats such as PNG, JPG, HEIC, and TIFF are supported.

What export formats are available?
You can copy extracted tables directly or export them as CSV or Excel files.
